# matlab-kalshi v0.1.0

Initial release of the MATLAB interface for the Kalshi Trade API.

## Highlights

- REST market data, normalized response-list helpers, and table APIs.
- Authenticated portfolio reads for balance, positions, orders, fills, and API limits.
- Demo-safe trading workflows for create/cancel, batch cancel, and order lookup helpers.
- WebSocket subscribe, reconnect/resubscribe, normalized event parsing, and orderbook stream state.
- Reproducible `.mltbx` packaging through `buildtool package`.

## Validation

- Offline MATLAB unit tests.
- Live Kalshi demo balance check.
- Live Kalshi demo post-only order create/cancel check.
